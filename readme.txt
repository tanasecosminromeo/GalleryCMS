GalleryCMS is a free, open source photo gallery CMS. 

This repo is for the development of GalleryCMS 2.0 which includes these high-level updates:
  - Upgrade the underlying CodeIgniter framework to 2.0.2
  - Implement HMVC
  - Update security & authentication
  - Build out additional features for album/asset management
  - Create user management
  - Upgrade installer

Installation
  - Create a database and update config/database.php
  - Un-archive this project into your webserver
  - Enter http://{myprojectname}/install
